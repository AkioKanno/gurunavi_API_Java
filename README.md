本サンプルは、ぐるなび社にて提供されているレストラン検索の WebA PI をJava から
コールするサンプルです。

日曜プログラマー向けの簡単サンプルです。

(エラー処理などの記載は簡単サンプルにならないので、ご自分でハンドリングお願いします)

サンプルでは緯度、経度、範囲などは固定値で入れてますが、引数などからもらうように
カスタマイズして利用してください。

また、URI組み立て時に 返却を XML にしているので、Json で扱いたい人は
  format=json
と変更してください。


※APIアクセスキーがない方は、以下から

http://api.gnavi.co.jp/api/use.html

「ユーザー登録を行う」選択から登録できます
